# Component Schemas for Spectrum Designers

A Slidev presentation explaining the benefits and value of designers working with Spectrum component schemas.

## Context

This presentation is part of the Doc Squad Data Group meeting, specifically for the "Vision for data and design authoring" agenda item led by Garth Braithwaite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the displayed URL (typically `http://localhost:3030`)

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run export` - Export to static files
- `npm run export-pdf` - Export to PDF

## Presentation Topics

- What are component schemas?
- Designer workflow integration

- Current pain points
