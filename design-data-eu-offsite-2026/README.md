# Design Data - EU Offsite

A Slidev presentation for the Design Data EU offsite, using the `theme-apple-basic` theme.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3030`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run export` - Export to static files
- `npm run export-pdf` - Export to PDF

## Presentation Structure

The presentation covers:
- The legacy world: hyphen-delimited token names
- The fix: the name object
- The registry: keeping vocabulary consistent
- Anatomy overlap: component parts vs. token names
- Monorepo architecture (mermaid diagram)
- Speed: the SDK's embedded database
- Platform autonomy: `spec/manifest.md`
- Tools: Figma variables, AI tooling, CLI, TUI
- What's next: roadmap & Q&A

See `outline.md` for the source talking-points outline (also kept in Obsidian at
`Projects/DNA/Design Data Offsite Presentation.md`).

## Theme

This presentation uses the `@slidev/theme-apple-basic` theme for a clean, Apple-inspired design.
