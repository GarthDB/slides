# Spectrum Design Data - Onsite Shareout

A Slidev presentation for the Spectrum Design Data onsite shareout using the `theme-apple-basic` theme.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3030`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run export` - Export to static files
- `npm run export-pdf` - Export to PDF

## Presentation Structure

The presentation covers:
- Design Data Complexity
- Engineering Strategy
- Component API Progress
- Token Workflow
- Reducing Complexity

## Theme

This presentation uses the `@slidev/theme-apple-basic` theme for a clean, Apple-inspired design.
